# realty
부동산 (Real Estate)

## 🌐 GitHub Pages로 공개된 웹사이트

이 프로젝트는 HTML과 JavaScript만으로 구성된 부동산 정보 웹사이트입니다.

### 📖 GitHub Pages 설정 방법

1. **GitHub 저장소 설정**
   - 저장소의 Settings 탭으로 이동
   - 왼쪽 메뉴에서 "Pages" 선택
   - Source 섹션에서 브랜치 선택 (예: `main` 또는 `copilot/add-public-html-page`)
   - 루트 디렉토리 `/` 선택
   - Save 버튼 클릭

2. **웹사이트 접속**
   - 설정 후 몇 분 후에 `https://mosframe.github.io/realty/`에서 접속 가능

### 🎯 주요 기능

- ✨ 순수 HTML, CSS, JavaScript로 구현
- 🏠 부동산 매물 추가 및 관리
- 📱 반응형 디자인
- 🎨 모던한 UI/UX

### 💻 로컬에서 실행

```bash
# 간단히 index.html 파일을 브라우저에서 열기
# 또는 로컬 서버 실행 (Python 예시)
python -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```
